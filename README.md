# travel_microservice

A microservice-based architecture for a travel application.

## Structure
- `gateway/` - API Gateway
- `services/`
  - `auth-service/`
  - `user-service/`
  - `destination-service/`
  - `booking-service/`
  - `recommendation-service/`
  - `notification-service/`
- `database/`
- `docker-compose.yml`
